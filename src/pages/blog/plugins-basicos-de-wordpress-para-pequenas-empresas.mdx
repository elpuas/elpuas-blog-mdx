Si bien WordPress es sin duda la solución web adecuada para cualquier pequeña empresa, no es la solución exacta para su pequeña empresa tal y como está. Ahí es donde los plugins entran en juego. Los plugins de WordPress permiten a los propietarios de pequeñas empresas ampliar, extender y personalizar WP, Ajustandolo precisamente para las necesidades específicas del sitio web de su empresa.

![Artboard 1](https://www.elpuas.com/wp-content/uploads/2018/04/Artboard-1.png)

A partir de la publicación de este post, actualmente hay 54.884 plugins disponibles para descargar en el repositorio de plugins de WordPress. Ahora, aunque su sitio web no necesita tantos plugins (por favor, no descargue e instale 54.884 complementos, ¡eso sería horrible!),  Hay ciertos plugins que recomiendo para todos los sitios web de pequeñas empresas. Los desglose en las siguientes funciones:

## Optimización de motores de búsqueda (SEO)

No tiene sentido tener un sitio web si no está generando tráfico hacia él, por lo que es importante que practique continuamente las estrategias de optimización de motores de búsqueda (SEO). Con una sólida calificación de cinco estrellas, [Yoast SEO](https://wordpress.org/plugins/wordpress-seo/) es uno de los plugins más utilizados y confiables para SEO. También es muy fácil de usar. Como propietario de una pequeña empresa, está obligado a hacer malabares con muchos roles. Probablemente no desee agregar un especialista en SEO a esa lista. Al instalar Yoast en su sitio web de WordPress, no tendrá que hacerlo. Hace el SEO pesado por ti. Concéntrate en crear contenido y deja que Yoast te guíe desde allí. Yoast SEO ofrece tanto un complemento premium como gratuito, con la versión premium te ofrece más funciones.

## Formularios

![user-friendly-contact-form-tips](https://www.elpuas.com/wp-content/uploads/2018/04/user-friendly-contact-form-tips.jpg)

Ya sea que desee capturar los datos de los clientes o dar a los visitantes de su sitio web una forma de dejarle un mensaje o comentario, querrás un formulario en el sitio web de tu pequeña empresa para lograrlo. Recomiendo [Gravity Forms](https://www.gravityforms.com/).  Desde formularios simples a complejos, este plugin de WordPress se puede extender aún más con extensiones de comercio electrónico, que le permite vender productos digitales, suscripciones o registro de eventos. Tenga en cuenta que Gravity Forms es solo un plugin premium, sin una versión gratuita disponible para descargar. Sin embargo, existen diferentes niveles de precios ofrecidos, por lo que puede encontrar la versión correcta que se ajuste a su presupuesto.

Consejo: Si está buscando un plugin de formulario de contacto gratuito, le recomiendo el Formulario de [Caldera Forms](https://wordpress.org/plugins/caldera-forms/) .

## Analytics

¿Qué elementos en su sitio web están convirtiendo con éxito a sus visitantes? ¿Qué elementos no son? ¿Qué necesita para mejorar? La mejor manera de encontrar las respuestas es haciendo un seguimiento de analytics del sitio web. Con [MonsterInsights](https://www.monsterinsights.com/), puede realizar un seguimiento de una gran cantidad de estadísticas de sitios web, incluyendo de dónde vienen sus visitantes, cuál es su comportamiento cuando están en su sitio, e incluso qué páginas les gusta ver. Tenga en cuenta que deberá tener una cuenta de [Google Analytics](https://analytics.google.com/) configurada primero para poder beneficiarse de MonsterInsights, pero una vez que lo haga, se abrirá todo un mundo de estadísticas para usted. MonsterInsights ofrece una versión premium con múltiples niveles de precios, pero también puede descargar la versión gratuita [aquí](https://wordpress.org/plugins/google-analytics-for-wordpress/).

## Optimización de imágenes

Los usuarios de Internet quieren actuar rápido y esperar resultados rápidos. Para ellos, un sitio web lento es indicativo del servicio al cliente de la empresa. Si su sitio web se está ejecutando lentamente, el tamaño de las imágenes cargadas podría tener mucho que ver con eso. El tamaño importa y las imágenes grandes a menudo son los culpables de sitios web lentos. Comprima sus imágenes con el complemento [Smush Image](https://wordpress.org/plugins/wp-smushit/). No perderá ninguna calidad de imagen y la velocidad de su sitio web mejorará. ¡Lo mejor de todo es que Smush Image es gratis!
