El sitio web móvil frente a una aplicación móvil puede llegar a ser un debate acalorado.

Muchas personas sienten que no necesita una aplicación móvil, solo necesita un sitio web que se vea bien en los dispositivos móviles. Otros dicen que las aplicaciones móviles tienen beneficios que un sitio web no puede proporcionar. ¿Quién tiene la razón?

Todos necesitan un sitio web responsive que se vea bien en todos los dispositivos móviles. No es realmente tan difícil obtener un sitio web responsive, la mayoria de plantillas en el repositorio de WordPress son responsive.

Los usuarios van a ver su sitio web en sus dispositivos, y usted debe respaldar eso. Es obvio que necesita un sitio web responsive, pero ¿necesita también  una aplicación móvil? ¿Es una aplicación móvil mejor que su sitio web móvil?

Veamos que es mejor para su negocio, si una aplicación móvil o un sitio web móvil .

### Pros y contras de sitios web móviles

Un sitio web móvil generalmente no es un sitio web separado, es un diseño receptivo que funciona para todos los tamaños de pantalla. Los visitantes móviles deben poder acceder a la mayoría del contenido que tiene en su sitio web completo, pero de una manera que esté optimizada para pantallas más pequeñas.

Creo que todos deberían tener un sitio web móvil, por lo que estos pros / contras son solo una comparación con una aplicación móvil.

Pros.

- Económico, generalmente incluido en el precio de diseño de su sitio web
- Funciona en todos los dispositivos: no necesita un sitio aparte para iOS / Android
- Fácil de configurar: no enviar a tiendas de aplicaciones, solo necesita un dominio y alojamiento

Contras

- Sin notificaciones push
- Sin acceso fuera de línea
- Sin presencia en la tienda de aplicaciones

Un diseño pobre puede causar problemas como:

- Problemas de desempeño
- Diseño desordenado
- Poca usabilidad

Los sitios web móviles bien diseñados pueden ser un gran activo comercial. Como casi 1/3 del tráfico de Internet es móvil, es obvio que lo necesita.

No son costosos de configurar, están disponibles en todos los dispositivos y son fáciles de mantener y actualizar.

### Donde los sitios web móviles fallan

He visitado demasiados sitios web en mi teléfono que estaban mal optimizados para dispositivos móviles y me hicieron ir inmediatamente a otro sitio. Algunos sitios móviles pueden estar llenos de anuncios, ventanas emergentes y contenido que no se necesita ver en el dispositivo móvil.

Los sitios web móviles también pueden tener una optimización excesiva, tratando de que se vea y se sienta como una aplicación nativa. Esto generalmente empeora las cosas, con animaciones entrecortadas y diseños sobre diseñados. Hay muchos ejemplos divertidos de personas que arruinaron sus sitios web móviles en el sitio [wtfmobileweb](http://wtfmobileweb.com/).

Un sitio móvil bien diseñado puede encargarse de muchos de estos problemas, pero aún no puede hacer las cosas que una aplicación móvil puede hacer.

Echemos un vistazo a las ventajas y desventajas de una aplicación móvil.

### Pros y contras de aplicaciones móviles

Las aplicaciones móviles pueden hacer cosas que los sitios web no pueden hacer, como las notificaciones automáticas y el acceso sin conexión, pero también tienen inconvenientes. No son necesarios para algunas personas, por ejemplo, si usted es un blogger ocasional probablemente no necesite una aplicación. Sin embargo, muchas empresas pueden beneficiarse de una, veamos los pros y los contras.

Pros

- Notificaciones push: envía mensajes de texto a tus suscriptores
- Acceso sin conexión: lea artículos, escuche podcasts o explore un catálogo de productos mientras está en un avión.
- Gran rendimiento: una aplicación bien diseñada puede ser más rápida que un sitio web
- Menos desordenado: esto depende del diseño del sitio web, pero las aplicaciones tienden a tener menos cosas
- Acceda a las Apps Stores: es otra forma de ponerse delante de sus clientes, también obtiene un botón en su pantalla de inicio.

Contras

- Gasto adicional: aunque algunas aplicaciones son asequibles ($ 49 / mes por ejemplo), sigue siendo otro costo
- Configuración adicional: debe configurar la aplicación y enviarla a las tiendas de aplicaciones.

Las aplicaciones móviles pueden ayudar a su público a acceder a su contenido más rápido y mantenerlo mejor. Enviar notificaciones automáticas es una gran ventaja, ya sea que esté publicando artículos o vendiendo productos.

Las aplicaciones son mejores para acceder al contenido sin conexión, como escuchar audio o leer un artículo en un avión.

Pueden costar tiempo y dinero extra, y no todas las empresas están preparadas para eso.

### El ganador

El ganador depende de su negocio. Para el blogger casual, el sitio web móvil gana. Para una empresa como Instagram, la aplicación móvil gana.

Para muchas empresas, el ganador es una combinación de ambos. Una estrategia móvil en dos frentes puede aprovechar tanto un sitio móvil como una aplicación para una distribución máxima de su contenido.

Un ejemplo sería una tienda en línea. ¿Por qué no dejar que los visitantes compren a través de su sitio web, así como una aplicación? Los restaurantes pueden ofrecer menús actualizados, direcciones y pedidos en línea. Las revistas pueden enviar notificaciones automáticas cuando se publican nuevos artículos.

En lugar de uno u otro, considere incorporar ambos en su estrategia móvil.
